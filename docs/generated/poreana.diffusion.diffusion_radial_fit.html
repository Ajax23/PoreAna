    diffusion_radial_fit¶

    poreana.diffusion.diffusion_radial_fit(link, len_step=None)¶

    This function use the diffusion profiles over the bins from the function do_mc_cycles() to caluclate the final diffusion coefficient. For that a line is fit of the mean diffusion \(D_{mean}({\Delta_{ij}t_{\alpha}})\) for the different lagtimes as a function of \(1/\Delta_{ij}t_{\alpha}\) . The function plot all mean diffusion \(D_{mean}({\Delta_{ij}t_{\alpha}})\) over all inverse input lagtimes and the linear fit. Additional the final mean diffusion coefficient \(D\) for a lagtime \(\Delta_{ij}t_{\alpha} \rightarrow \infty\) is printed. Also a table of the selected step length and the belonging \(D_{mean}({\Delta_{ij}t_{\alpha}})\) and the \(D_{avg}({\Delta_{ij}t_{\alpha}})\) can be displayed

    Formel für avg falls das drin bleibt

    Parameters:
    link: string

    link to the diffusion data object generated by the sample routine :func: sample_box

    len_step: integer list, optional (default = None)

    list of the different step length, if it is None the lag times which are contains in the obj file are printed
    Returns:
    diffusion: float

    diffusion coefficient \(D \ (\frac{m^2}{s})\) for the calculated system

    diff_table : obj

    table of used lag times for fitting of \(D \ (\frac{m^2}{s})\)
