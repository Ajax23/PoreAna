    poreana.diffusion¶

    Analyse diffusion in a pore.

    Bin Diffusion¶

    Analysis

    bins(data_link[, ax_area, intent, is_norm]) This function calculates the axial (z-axis) diffusion coefficient as a function of the radial distance.
    cui(data_link[, z_dist, ax_area, intent, …]) This function samples and calculates the diffusion coefficient of a molecule group in a pore in both axial and radial direction, as described in the paper of Cui.

    MC Diffusion¶

    Analysis

    diffusion_fit(link[, len_step, is_std]) This function uses the diffusion profiles over box length which are calculated in the function do_mc_cycles() to estimate the final diffusion coefficient.
    diffusion_pore_fit(link_pore, link[, …]) This function uses the diffusion profiles in the pore area which are calculated in the function do_mc_cycles() to estimate the final diffusion coefficient.
    diffusion_radial_fit(link[, len_step]) This function uses the diffusion profiles over box length which are calculated in the function do_mc_cycles() to estimate the final diffusion coefficient.

    Profiles

    diff_profile(link[, len_step, infty_profile]) This function plots the diffusion profile for a infinity lag time (\(\Delta_{ij}t_{\alpha} \rightarrow \infty\)) over the box fitted with the specified \(\mathrm{len}\_\mathrm{step}\) list.
    diff_radial_profile(link[, len_step, avg]) This function plots the radial diffusion profile for a infinity lag time (\(\Delta_{ij}t_{\alpha} \rightarrow \infty\)) over the box fitted with the specified \(\mathrm{len}\_\mathrm{step}\) list.
    diff_pore_profile(link_pore, link[, …]) This function plots the diffusion profile in the pore areas for a infinity lag time (\(\Delta_{ij}t_{\alpha} \rightarrow \infty\)) over the box fitted with the specified \(\mathrm{len}\_\mathrm{step}\) list.
    df_profile(link[, len_step]) This function plots the free energy profile over the box for the calculated lag times.

    Input & Output Tables

    plot_trans_mat(link_in, step[, kwargs]) This function plots the occupation of a normalized transition matrix as a heatmap.

    print_model_inputs(link_out[, print_con]) This function prints the model inputs of the calculation.
    print_statistics_mc(link_out[, print_con]) This function prints the statistic of the MC Run for every lag time.
    print_coeff(link_out[, print_con]) This function prints the final coefficients of the profile for every lag time profile.
